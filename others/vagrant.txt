Vagrant:跨平台的开发环境,github.com/apanly/mooc
	1.安装: VituralBox,Vagrant,Xshell
	2.vagrant 命令
		vagrant box list // 查看目前已有的box
 		vagrant box add // 新增加一个box
		vagrant box remove // 删除指定box	
		vagrant init 	// 初始化配置vagrantfile
		vagrant up	// 启动虚拟机
		vagrant ssh	// ssh登录虚拟机
		vagrant suspend // 挂起虚拟机 
		vagrant reload // 重启虚拟机
		vagrant halt // 关闭虚拟机
		vagrant status // 虚拟机状态
 		vagrant destroy // 删除虚拟机
	3. 
		1. 端口转发:config.vm.network"forwarded_port", guest:80,host:8888	
		2. 共享目录:config.vm.synced_folder "/users/vincent/code/","/home/www",:nfs => true
		3. 网络配置:config.vm.network "private_network","192.168.199.101"
